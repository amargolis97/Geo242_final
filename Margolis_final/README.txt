This is the readme doc for the final project. Attached in this folder are 9 files, including:

readme.txt
- this file which has instructions and describes what each file is in the Margolis_final directory

query.csv
- this is the data file from USGS (see reference on final report for source)

Geo242_final.ipynb 
- this is the file that process the data. It is compatible for Jupyter Notebook. 

Geo242_final_report.pdf
- this is the final, clean pdf copy of the report with figures, citations, references, and equations

Margolis_Final.tex
- this is the latex file script used to create the pdf version of the final report

references.bib
- this is the references file used in Margolis_Final.tex

California_map.png
- this is the overview map of California, showing where the study area is and other reference points. 
- the script for this map can be found in the Jupyter Notebook file: Geo242_final.ipynb

Catalog_map_Parkfield.png
- this is a map of all 2003 events from the USGS catalog.
- the script for this map can be found in the Jupyter Notebook file: Geo242_final.ipynb

Best_fit_line_Parkfield.png
- this is a map of all 2003 events from the USGS catalog, that are color coded for depth
- also on this map is included the best fit line, plotted with the epicenter locations marked in magenta
- the script for this map can be found in the Jupyter Notebook file: Geo242_final.ipynb
